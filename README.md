# Telegram Group Chat Agent

## Overview

This project is a Telegram Group Chat AI Agent built using **n8n automation** and powered by the **Google Gemini Chat Model**.
The workflow automatically receives messages from a Telegram group, processes them through an AI agent, and sends intelligent responses back to the chat.

---

## Workflow Structure

### Components Used

* **Telegram Trigger**

  * Detects incoming Telegram group messages.

* **Edit Fields**

  * Processes or formats incoming message data before sending it to the AI agent.

* **AI Agent**

  * Handles user interaction and generates AI-powered responses.

* **Google Gemini Chat Model**

  * Provides intelligent conversational capabilities.

* **Simple Memory**

  * Stores previous conversation context for improved continuity.

* **Send a Text Message**

  * Sends AI-generated replies back to the Telegram group.

## Installation & Setup

### Requirements

* n8n installed
* Telegram Bot Token
* Google Gemini API Key

### Steps

1. Import the workflow JSON into n8n
2. Configure Telegram credentials
3. Configure Google Gemini API credentials
4. Activate the workflow
5. Add the bot to your Telegram group
6. Start chatting with the AI agent
